sq1-inventory
=============

square one inventory app back end

This app is an app to track inventpry and to generate shopping list.

# How to:

## Setup environment

`make virtualenv`

## install package

`make requirements`

## test
`make test`

## deployment
`git push heroku master`

# Environment Variables
- `SQ1_ENV`: environment (dev or prod)

# Rest
- `/items`: get list of items
